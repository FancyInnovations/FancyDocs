## Permission Patterns

FancyCore uses a hierarchical permission system. Most permissions follow the pattern `fancycore.commands.<category>.<action>`.

## Chat Permissions

### Chat Room Access
- `fancycore.chatrooms.<name>` - Access to a specific chat room (replace `<name>` with the chat room name)

### Chat Bypass Permissions
- `fancycore.chat.bypassmute` - Bypass chat room mute restrictions
- `fancycore.chat.bypasscooldown` - Bypass chat cooldown restrictions

### Chat Commands
- `fancycore.commands.message` - Send private messages
- `fancycore.commands.reply` - Reply to private messages
- `fancycore.commands.ignore` - Ignore players
- `fancycore.commands.uningnore` - Unignore players
- `fancycore.commands.togglemessages` - Toggle receiving private messages
- `fancycore.commands.chatrooms.info` - View chat room information
- `fancycore.commands.chatrooms.list` - List chat rooms
- `fancycore.commands.chatrooms.create` - Create chat rooms
- `fancycore.commands.chatrooms.delete` - Delete chat rooms
- `fancycore.commands.chatrooms.watch` - Watch chat rooms
- `fancycore.commands.chatrooms.unwatch` - Unwatch chat rooms
- `fancycore.commands.chatrooms.watching` - View watched chat rooms
- `fancycore.commands.chatrooms.switch` - Switch chat rooms
- `fancycore.commands.chatrooms.clearchat` - Clear chat
- `fancycore.commands.chatrooms.mute` - Mute chat rooms
- `fancycore.commands.chatrooms.unmute` - Unmute chat rooms
- `fancycore.commands.chatrooms.cooldown` - Set chat room cooldown
- `fancycore.commands.chatcolor.set` - Set chat color
- `fancycore.commands.broadcast` - Broadcast messages
