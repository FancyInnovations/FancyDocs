# Commands for the chat system

<Callout type="warn">
    The command list is not complete and not all commands are properly documented yet.
    Some commands might not be implemented yet.
</Callout>

## Messages

### message

Sends a private message to the specified player.
- **Permissions**: `fancycore.commands.message`
- **Syntax**: `/message <player> <message>`
  - `<player>` - Target player name
  - `<message>` - Message content
- **Alias**: `/msg`

### reply

Replies to the last private message received.
- **Permissions**: `fancycore.commands.reply`
- **Syntax**: `/reply <message>`
  - `<message>` - Reply content
- **Alias**: `/r`

### messagespy

Toggles message spying to see private messages between other players.
- **Permissions**: `fancycore.commands.messagespy`
- **Syntax**: `/messagespy`
- **Alias**: `/msgspy`

### togglemessages

Toggles the ability to receive private messages.
- **Permissions**: `fancycore.commands.togglemessages`
- **Syntax**: `/togglemessages`
- **Alias**: `/togglemsg`

## Chat rooms

### chatroom watch

Watch a specific chat room. You will receive messages from this room.
- **Permissions**: `fancycore.commands.chatroom.watch`
- **Syntax**: `/chatroom watch <name>`
  - `<name>` - Chat room name

### chatroom unwatch

Unwatch a specific chat room.
- **Permissions**: `fancycore.commands.chatroom.watch`
- **Syntax**: `/chatroom unwatch <name>`
- `<name>` - Chat room name

### chatroom list

Lists all available chat rooms.
- **Permissions**: `fancycore.commands.chatroom.list`
- **Syntax**: `/chatroom list`

### chatroom switch

Switches to a specific chat room. Your messages will be sent to this room.
- **Permissions**: `fancycore.commands.chatroom.switch`
- **Syntax**: `/chatroom switch <name>`
  - `<name>` - Chat room name

### chatroom clearchat

Clears the chat for a specific chat room.
- **Permissions**: `fancycore.commands.chatroom.clearchat`
- **Syntax**: `/chatroom clearchat <name>`
  - `<name>` - Chat room name
- **Alias**: `/clearchat` (clears current chat room)

### chatroom cooldown

Sets a cooldown for sending messages in a specific chat room.
- **Permissions**: `fancycore.commands.chatroom.cooldown`
- **Syntax**: `/chatroom cooldown <name> <seconds>`
  - `<name>` - Chat room name
  - `<seconds>` - Cooldown in seconds
- **Alias**: `/chatcooldown` (sets cooldown for current chat room)

### chatroom mute

Mutes a specific chat room, preventing players from sending messages.
- **Permissions**: `fancycore.commands.chatroom.mute`
- **Syntax**: `/chatroom mute <name>`
  - `<name>` - Chat room name
- **Alias**: `/mutechat` (mutes current chat room)

### chatroom unmute

Unmutes a specific chat room, allowing players to send messages again.
- **Permissions**: `fancycore.commands.chatroom.unmute`
- **Syntax**: `/chatroom unmute <name>`
  - `<name>` - Chat room name
- **Alias**: `/unmutechat` (unmutes current chat room)
