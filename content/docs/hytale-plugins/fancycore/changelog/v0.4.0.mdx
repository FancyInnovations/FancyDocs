# FancyCore v0.4.0

- Moderation system
- New commands: `/kick`, `/warn`, `/mute`, `/tempmute`, `/unmute`, `/ban`, `/tempban`, `/unban`
- Punishment management: `/punishments list`, `/punishments listactive`, `/punishments info`
- Report system: `/report`, `/reports list`, `/reports info`, `/reports resolve`
