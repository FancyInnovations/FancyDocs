<Callout>
FancySpaces is a platform every software project needs. It provides a wide range of features to help you manage your project, distribute your artifacts, and track your analytics.
</Callout>

## Features

- Distribute artifacts (executables, jar files ...) to users
- Publish maven artifacts (for example libraries)
- View Javadoc from published maven artifacts
- Create issues and visualize them on a kanban board
- Sync issues from GitHub
- Publish blog articles and view them publicly
- Store secrets (database credentials, API keys ...) securely
- Track analytical data (metrics, events, logs) and visualize it on dashboards
